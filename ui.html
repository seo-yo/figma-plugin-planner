📘 Plan Doc Builder 사용법

① 빠른 시작

  • 상단 탭 선택: PC 퍼블 / PC URL / MO 퍼블 / MO URL
  • 행에 content / url 입력 (또는 "빈 행 N개 생성"으로 빠르게 시작)
  • 표 생성 클릭 → Figma에 표 자동 생성

② 데이터 입력 팁

  • content에 여러 줄 붙여넣기 → 줄 단위로 자동 분배
  • 퍼블 탭 content는 드롭다운 선택 가능 (슬라이드/페이드인/탭/영상 등)
  • URL은 풀 URL 붙여넣어도 /product-detail/... 형식으로 자동 정리
  • 🔎 아이콘 → MLB Korea 검색 결과 새 창으로 열림

③ 행 추가/삭제

  • 행 사이 호버 → 파란 + 클릭으로 그 위치에 삽입
  • 휴지통 아이콘으로 행 삭제
  • 행 변경 시 캔버스의 자유 마커도 자동 재라벨링

④ 표 ↔ Figma 양방향

  • 표 생성: 빌더 데이터 → Figma 표
  • 가져오기: Figma 표 → 빌더로 역동기화
  • 탭 전환 시 자동으로 가져오기 실행

⑤ 마커 배치

  • ▶ 마커 생성: 1번부터 순서대로, Figma 요소 클릭마다 다음 라벨 배치
  • N부터 마커 생성: 중간부터 시작 (예: C / 8)
  • 셀 No 라벨을 직접 클릭 → 디자인 요소 클릭으로도 단일 배치 가능
  • 이미 모두 배치되어 있으면 재생성 confirm

⑥ 마커 색상 (탭별 자동)

  • PC 퍼블 — 주황 #FF8400
  • PC URL — 하늘 #7BCCFF
  • MO 퍼블 — 핑크 #E83A8C
  • MO URL — 민트 #4ABDAC

⑦ Figma 자동 배치 좌표

  • PC 표: x=5096 / MO 표: x=11500
  • 퍼블: y=453 / URL: y=2917

⑧ 주의사항

  • 표 생성 시 기존 행 모두 새 데이터로 교체됨
  • 탭 전환 시 진행 중인 마커 배치는 자동 중단
  • 마커 모두 삭제는 자유 위치 마커만 (표 안 셀 No는 유지)
